【GitHubへの反映方法】

1. このZIPを展開します。
2. 「whiskey」フォルダーの中身をすべて選択します。
3. GitHubの「whiskey」リポジトリを開きます。
4. Add file → Upload files を選び、フォルダーの中身をアップロードします。
5. Commit changes を押します。
6. GitHub Pagesの更新後、ブラウザで Ctrl＋F5 を押します。
