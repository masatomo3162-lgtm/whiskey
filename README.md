# Whisky Cabinet v1.0.2

ウイスキーのコレクション、複数人レビュー、フレーバー、価格帯、レア度、
開栓中10本、次に開けるボトルの抽選、購入予定5選を管理できる
単体HTML形式のブラウザアプリです。

## 特徴

- インストール不要
- 外部ライブラリ不要
- オフライン動作
- データはブラウザの localStorage に保存
- JSONバックアップ・復元
- CSV出力
- スマートフォン・タブレット対応
- 対応端末ではカメラ撮影からボトル写真を登録可能
- ボトル写真を最大1200px・高画質で保存
- 一覧・詳細・抽選画面で写真全体を切り抜かず表示

## GitHub Pagesで公開する方法

1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内の `index.html` と `README.md` をアップロードします。
3. リポジトリの `Settings` を開きます。
4. 左側の `Pages` を選びます。
5. `Build and deployment` の Source を `Deploy from a branch` にします。
6. Branchを `main`、フォルダを `/(root)` にして保存します。
7. 数分後に表示される公開URLへアクセスします。

## データ保存について

登録データは、利用端末・利用ブラウザごとの localStorage に保存されます。
GitHub Pagesへ公開しても、利用者同士でデータは共有されません。

ブラウザのデータ削除や端末変更に備えて、アプリ内の
「バックアップ保存」を定期的に使用してください。
